AIP Dashboard

The AIP Dashboard (Publication Dashboard) is the central management interface for the eAIP (Electronic Aeronautical Information Publication) module. It provides AIS personnel with tools to manage batch dates, documents, and publication cycles.


Upload File

The Upload File Button navigates to the file upload interface

Archives

The Archives Button is to view the archived documents


Batch Dates Management

Purpose:

Batch dates define publication cycles for AIP documents. Each batch has:

- Effective Date – When documents become active (UTC)

- Publication Date – When documents are published (UTC)

- Self-Expiry Date – Optional automatic expiration date

- Status – CURRENT, UPCOMING, EXPIRED, or INACTIVE

Batch Date Form

Form to create new batch dates with fields:

- Batch Type

- Effective Date (UTC datetime picker)

- Publication Date (UTC datetime picker)

- Description

- Self-Expiry Date (optional)


Show Batch Dates

Toggles visibility of batch dates table

Update Batch Statuses

Manually triggers status recalculation for all batch dates and associated files


Document Filtering

Filter by issuance type and categories


Refresh File Statuses Button

Button to manually refresh document statuses based on the current date/time


Documents Table and Search Functionality

Searchable table with columns, actions (View, Archive, Delete), and keyboard shortcuts

- Search Input – Real-time filtering as you type

- Field Selector – Search in All Fields, Document ID, Section, Category, Sub-Section, Title, or Status

- Keyboard Shortcut – `Ctrl+F` / `Cmd+F` focuses search input

- Result Count – Shows number of matching documents


Document Actions

View

opens document viewer

Archive/Unarchive

toggles archive status

Delete

Removes document (with confirmation)
