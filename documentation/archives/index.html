Archives

A simple interface to view and restore archived AIP documents (AIP, AMDT, SUP, AIC) so AIS staff can retrieve and unarchive previously archived files.

What you see

- A filter dropdown to show all categories or only one category (AIP, AMDT, SUP, AIC).

- A color legend showing which color matches each category (blue = AIP, green = AMDT, yellow = SUP, red = AIC).

- A table of archived files with columns:

  - Label — the document name/subsection

  - Category — the document type (with a colored dot)

  - Issuance Date — the document’s issuance/effective date

  - Access PDF — actions: View PDF (open file) and Unarchive (restore file)

Step-by-step (How to use it)

1. Open the Archives page from the AIP dashboard.

2. Use Filter by Category to narrow the list to a specific type (optional).

3. Click a column header to sort the table by that column. The page defaults to sorting by Issuance Date (newest first).

4. To view a document, click View PDF — the PDF opens in a new tab from storage.

5. To restore a file back to active lists, click Unarchive; this sends a request to the server to toggle the archive status.

Quick tips

- Use the filter to reduce the list if the archive is large.

- After unarchiving, check the AIP dashboard or relevant module to confirm the file is back in the active list.

- If a PDF does not open, the file may be missing from storage or the storage link may be incorrect.
